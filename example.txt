  Setup Steps

  1. cd emoji-site && npm install
  2. npx convex dev — creates Convex project, generates _generated/ files
  3. Copy the deployment URL from the terminal and replace https://YOUR_DEPLOYMENT.convex.cloud in
  index.html
  4. In Convex dashboard: set UPLOAD_PASSWORD environment variable
  5. Open index.html and test the upload flow
  6. npx convex deploy when ready for production
